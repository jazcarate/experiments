Kotlin library to evaluate AB Testing variants

For EXP0001, split the users so that half are participating in the experiment, and the rest in the control variant
Becomes 
{
  "EXP001>": {
    "$children": {
      "participating": { "$rand": { "$gt": 0.5 } },
      "control": { "$always": true }
    }
  }
}
